# Stat

A Stat is an integer in-game character attribute. These can be base stats (e.g., _Strength_, _Vitality_), derived stats (e.g., _Health_, _DPS_), or qualifications (e.g., _Has Item_).

### Properties
* Name (`string`)
* Description (`string`)
* Value (`int`)

### Functions
* Set the Stat name, description
* Delete the stat
